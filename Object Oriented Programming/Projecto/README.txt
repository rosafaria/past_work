O código fonte pode ser encontrado na directoria Bib/src/bib
	
/*****-----*****-----*****-----*****-----*****-----*****/
Para executar o programa com uma JVM:
- Ter a certeza que estão na mesma directoria que o ficheiro jar os seguintes ficheiros:
	. admin
	. readers
	. books
	. DVDs
	. Reqs
	. pasta lib com o ficheiro jmathplot.jar dentro
 
 A biblioteca jmathplot é essencial para o funcionamento do programa, por isso se não estiver presente o programa não funcionará.
 
 Se os outros ficheiros, que funcionam como armazenamento da base de dados, não estiverem presentes, o programa gerará uma nova "base de dados" , em que apenas existirá o administrador "root" com password "root", e criará novos ficheiros para este efeito.
